# v36 derived quadratic algebra provenance report

## Question
Can the exact quadratic layer be introduced in a derived-only theory without postulating `sqrt(D)`?

## Decision
No free algebraic number is introduced. v36 treats the exact quadratic layer as a required but discriminant-indexed schema: `theta` is generated by the relation `theta^2 = D`, and concrete instantiation requires a proof-carrying operational discriminant source.

## Mathlib/reference status

- `QuadraticAlgebra` is a useful reference/wrapper candidate because it has explicit coordinates and multiplication.
